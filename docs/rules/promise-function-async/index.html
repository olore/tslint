--- ruleName: promise-function-async description: Requires any function or method that returns a promise to be marked async. rationale: |- Ensures that each function is only capable of 1) returning a rejected promise, or 2) throwing an Error object. In contrast, non-`async` `Promise`-returning functions are technically capable of either. This practice removes a requirement for consuming code to handle both cases. optionsDescription: Not configurable. options: null optionExamples: - 'true' type: typescript typescriptOnly: false requiresTypeInfo: true layout: rule title: 'Rule: promise-function-async' optionsJSON: 'null' ---
